# Newfront 2015
The DigitasLBi UK NewFront is a collaborative thought-leadership programme aimed at stimulating creativity within the branded content marketplace. 

## Prerequisites
[Node](https://nodejs.org/)
[Bower](http://bower.io/)
[Grunt](http://gruntjs.com/)


## install
```
bower install
```

## Develop
```
grunt server
```

## build
```
grunt dist
```
